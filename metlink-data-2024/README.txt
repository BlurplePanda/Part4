This folder contains original data downloaded from the Metlink site:
  https://www.metlink.org.nz/about/legal/general-transit-specification-feed/

This is in the  General Transit Feed Specification (GTFS) format, which is described
on the Google page:
  https://developers.google.com/transit/gtfs/reference?hl=en

This is 2024 data, and include timetable information.
Note that it is not the same as the files in the "data-full" folder, which was extracted from the 2022 metlink data.
